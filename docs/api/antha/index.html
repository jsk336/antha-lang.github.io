API references

Technical documentation

cmd/anthadoc

anthadoc automatically generates documentation for Antha protocols

cmd/anthafmt

Anthafmt formats Antha protocols

antha/ast

Package ast declares the types used to represent syntax trees for Antha packages.

anthalib/biotree

biotree provides a tree structure representing the tree of life

antha/build

Package build gathers information about Antha packages

antha/doc

anthadoc extracts source code documentation from an Antha AST

anthalib/execution

defines interface to the execution environment

antha/format

Package format implements standard formatting of Antha source.

anthalib/liquidhandling

defines types and methods for dealing with liquid handling requests

anthalib/mixer

defines interface to network calls to liquid handling to define semantics of mix()

anthalib/mixingrules

defines a structure for defining which kinds of liquids are miscible

antha/parser

Package parser implements a parser for Antha source files.

antha/printer

Package printer implements printing of AST nodes.

antha/scanner

Package scanner implements a scanner for Antha source text.

antha/token

defines constants representing the lexical tokens of the Antha programming language

anthalib/wconstraints

defines a structure for expressing and implementing constraints such as time and temperature

anthalib/wtype

defines core physical types representing substances, devices and labware

anthalib/wunit

anthalib/wutil

defines some utility functions
